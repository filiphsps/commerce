# commerce-marketing-common

Shared design primitives for the Nordcom Commerce marketing surfaces. Today this is
a single shared [Nordstar](https://www.npmjs.com/package/@nordcom/nordstar) `Theme`
re-used by `apps/landing` and `apps/admin` so the two surfaces look like they belong
to the same product.

For the canonical reference, see the [README on GitHub](https://github.com/filiphsps/commerce/blob/master/packages/marketing-common/README.md).
