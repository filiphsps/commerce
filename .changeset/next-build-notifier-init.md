Add `next-build-notifier`: a headless "new build available" indicator for Next.js. Generic build detection (any host) with first-class Vercel support, exposed as a config wrapper (`withBuildNotifier`), a route-handler factory (`createVersionRoute`), and a headless client API (`BuildNotifierProvider` / `useBuildNotification` / `<BuildNotifier>`).
