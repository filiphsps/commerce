Token Vertical · desktop Horizontal · any Any · mobile (< md) Cart drawer

Tokens added by this section
  • --product-card-cta-placement: float-pill | inline-button (default: float-pill for grid-friendliness)
  • --product-card-cta-pill-label: string (default: empty → icon-only)
  • --product-card-cta-pill-icon: string (default: "+")
  • --product-card-cta-inline-label: string (default: "Add to bag", i18n-driven)
  • --product-card-cta-inline-style: solid | ghost (default: solid)
  • --product-card-quick-add-presentation: auto | float | sheet (default: auto = surface routing rule)
