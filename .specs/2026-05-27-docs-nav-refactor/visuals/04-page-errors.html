API_UNKNOWN_LOCALE

An unknown or invalid locale was provided to the API.
UnknownLocaleError ↗ HTTP 400 ApiErrorKind

Possible causes

  • Invalid user input — a locale code that doesn't match the xx-XX BCP-47 pattern.
  • Misconfigured commerce provider — the requested locale isn't in the shop's i18n.locales set.
  • A previously-supported locale was removed from the shop record and a cached URL still references it.

How it's thrown

@nordcom/commerce-errors 
import { UnknownLocaleError } from '@nordcom/commerce-errors';

try {
  Locale.from('hello-world');
} catch {
  throw new UnknownLocaleError();
}
