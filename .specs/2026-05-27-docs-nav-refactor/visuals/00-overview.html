Implementation notes · Visuals are unbundled HTML for design review — final renders happen inside Fumadocs. Token names map directly onto Fumadocs CSS variables (--fd-background, --fd-primary, --fd-muted-foreground, etc.) — see spec §4 for the full mapping. The wordmark SVG is copied verbatim from apps/admin/public/logo.svg; no separate variant lives in the docs app.

Browse order · Start with 01-tokens.html to internalize the palette + type. Then walk through the four page templates (02 → 05) to see the system in use. 06–09 drill into details that recur across pages.
