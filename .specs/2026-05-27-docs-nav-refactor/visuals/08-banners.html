08 · Banners & callouts

Two families: JSDoc banners at the top of generated symbol pages, and MDX callouts inside hand-authored prose. Each tag/kind owns one color from the semantic palette — no overlap, no improvisation.

JSDoc tag banners · at top of symbol pages

Beta
Public API but the shape may still change without a major version bump. Pin a patch version if you depend on this in production.

Experimental
Behind the NEXT_PUBLIC_DRAFT_PREVIEW flag. Not enabled by default. API and behavior subject to change; do not depend on this in production.
@experimental · cyan dashed border. Stronger warning than beta — usually requires opt-in flags. Visible in dev only by default; production builds skip these pages unless a flag overrides.
Internal
This symbol is marked @internal and excluded from the Reference tab. It's surfaced here only when browsing source directly. Don't import outside the package.
@internal · gray, low contrast. Page is normally hidden — only generated when ?include=internal is set in build args. Sidebar omits it entirely. Search indexes only the title.

MDX callouts · inside prose

Concept
Tenant context is never implicit in this codebase. Every Shopify call constructs a fresh ShopifyApolloApiClient({ shop, locale }). New data-fetching helpers must take { shop, locale } in their arguments object.
Tip
For tenant-root reads that aren't entity-specific (e.g. sitemap indexes), use cmsTenantRootTags(shop) from @nordcom/commerce-cms/cache to get the matching fan-out tags. Don't hand-format tag strings.
Watch out
Never derive the shop from process.env or a singleton. The deployment serves many shops simultaneously; a wrongly-cached tenant context leaks data across shops.
Danger
dropTenantCache() sweeps every cached entity for the tenant — pages, articles, blocks, header, footer, business data, product metadata. Use only in tests or after a tenant-wide schema migration.
Example
See the storefront's app/[domain]/[locale]/blog/[slug]/page.tsx for the canonical pattern — await params, fetch article, return notFound() when null.
Note
The reserved Pages slug homepage is hard-coded — middleware rewrites /<locale>/ to /<locale>/homepage/ before routing. home or index won't work.
MDX callouts · five flavors, mapped to <Callout type="info|tip|warn|danger|example|note">. Each owns one semantic color from the palette. Left rail is thick (4.6px) for emphasis; outer border is thin (2.2px) to lighten the overall weight.

Inline pills · next to h1 / inline in prose

The new createCartSessionNew helper bootstraps a guest cart with the right tenant + locale wired up. The legacy initCartDeprecated path still works but emits a console warning. For draft-mode previews use useDraftPreviewExperimental — opt-in via the env flag described above.
Inline pills can hang next to any inline-code identifier — pulled from the same JSDoc tag data that drives the page-level banner. Keeps deprecation visible mid-prose, not just at page top.
